# school-front
semesta front adalah sebuah paket laman untuk frontend website sekolah. Paket ini dibuat berdasarkan studi kasus pada tempat kerja saya. Yaitu sebuah sekolah negeri di kabupaten Karawang.

## Fitur halaman
1. Landing Page
2. Single Page
3. Category Page
4. Article Page
5. Galeri
6. Static Page
7. Contact Page

## Teknologi yang digunakan
- HTML
- CSS
- Javascript

## Library/Framework yang digunakan
- Tailwind CSS
- jQuery
- jQueryUI
- slick.js
- SwiperJS
- SimpleLightbox.js

## Icon Library/Framework
- Bootstrap Icon
- IonIcon
- Font Awesome 4
